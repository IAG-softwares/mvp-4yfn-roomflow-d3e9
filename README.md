# RoomFlow

Efficient room and building space management system for organizations to handle bookings, avoid conflicts, and optimize space utilization

## Quick Start

```bash
# Install dependencies
npm run setup

# Start development servers
npm run dev
```

The frontend will be available at http://localhost:5173
The backend API will be available at http://localhost:3001

## Project Structure

├── frontend/          # React + Vite frontend

## Features

Real-time room availability checking, intuitive booking interface with calendar integration, automated booking confirmation and reminder notifications, booking modification and cancellation capabilities, conflict detection and resolution, user role management (admin vs regular users), dashboard analytics for space utilization, mobile-responsive design for on-the-go bookings, integration with popular calendar applications, and customizable notification settings for booking lifecycle events.

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Styling**: Tailwind CSS
